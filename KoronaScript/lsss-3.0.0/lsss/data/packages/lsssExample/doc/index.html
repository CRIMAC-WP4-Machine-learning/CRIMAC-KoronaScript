LSSS example package

This package contains examples on how to use some features of the LSSS API.

Python actions

This package defines actions demonstrating:

  • How to use the LSSS HTTP API from Python. (lsss.get, lsss.post)
  • How to use input parameters. (lsss.input)
  • How to use an SVG icon.

Web pages

These examples are web pages that use the LSSS HTTP API from JavaScript.

Python is not required.

  • misc.html: Miscellaneous examples.
  • map.html: Showing data from LSSS in a map.
  • chart.html: Plotting data from LSSS in a chart.
